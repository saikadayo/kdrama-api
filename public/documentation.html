K-Drama API Documentation

Business Logic URL HTTP Method Query Parameters Request body data format Response body data format
Get list of all movies /movies GET none none JSON movie array
Get data about a single movie by title /movies/:title GET title (URL) none JSON object with single movie information
Get data about a genre by title /genre/:name GET name (URL) none JSON object with genre name and description
Get data about a director by name /director/:name GET name (URL) none JSON object with director name, bio, and birth year
Allow new users to register /users POST none JSON object with user info (Username, Password, Email, Birthday) JSON object (created user)
Allow users to update their user info (username, password, email, date of birth) /users/:Username PUT Username (URL) JSON object with updated user info (Username, Password, Email, Birthday) JSON object (updated user)
Allow users to add movie to their favorite list /users/:Username/movies/:MovieID POST Username (URL), MovieID (URL) none JSON object (updated user with FavoriteMovies array)
Allow users to remove a movie from their favorite list /users/:Username/movies/:MovieID DELETE Username (URL), MovieID (URL) none JSON object (updated user with FavoriteMovies array)
Allow existing users to deregister /users/:Username DELETE Username (URL) none Confirmation message or error message
