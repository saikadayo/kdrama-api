Business Logic URL HTTP Method Query Parameters Request body data format Response body data format

Allow users to add movie to their favorite list /users/:id/:movieTitle POST user id (URL) JSON object with new movieTitle List of favorite movies updated
Allow users to remove a movie from their favorite list /users/:id/:movieTitle DELETE user id, movieTitle none Confirmation message and list of favorite movies updated
Allow existing users to deregister /users/:id DELETE user id (URL) none Confirmation message or error message
